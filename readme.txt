
# Ring Signature Demo

C++ tool to generate and verify ring signatures using RSA, showcasing anonymous signing.

## Features
- `sign`: Creates a ring signature for a message.
- `verify`: Checks the signatureâ€™s validity.

## Requirements
- OpenSSL library
- C++ compiler (e.g., g++)

## Build
```bash
make all
```
Executables appear in `bin/`.

## Usage
- **Sign**:  
  ```bash
  ./bin/sign  # Unix
  bin/sign.exe  # Windows
  ```
  - Enter signer (1 or 2) and private key (defaults: 107 for 1, 1233 for 2).

- **Verify**:  
  ```bash
  ./bin/verify  # Unix
  bin/verify.exe  # Windows
  ```
  - Outputs "True" or "False".

- **Clean**:  
  ```bash
  make clean
  ```

## Files
- `data/publickey.txt`: RSA keys (e1, n1, e2, n2, one per line).
- `data/message.txt`: Message to sign/verify.
- `data/signature.txt`: Generated by `sign`, used by `verify`.

## Notes
- Cross-platform (Linux/Windows) via Makefile.
- Store input files in `data/`.
- Originally an academic exercise, enhanced for demonstration.
